Tidak Ada Koneksi

BM Exam memerlukan koneksi internet untuk sinkronisasi data ujian, autentikasi, dan penyimpanan jawaban secara real-time.

⚠️ Penting saat mengerjakan ujian:

  • Pastikan koneksi internet stabil sebelum memulai
  • Jawaban tersimpan otomatis ke server
  • Hubungi pengawas jika koneksi terputus saat ujian
